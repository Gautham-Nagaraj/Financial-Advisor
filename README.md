# Financial Advisor

This agent analyzes your bank statements and provides advice based on the **Canada Revenue Agency (CRA)** guidelines.

> [!WARNING]
> **SECURITY & PRIVACY:** Do not upload statements containing your **full account numbers, name, or address**.
>  This app is for analysis purposes; please use a redacted or sample CSV file to protect your financial security.


### How to use:
1. Go to the project on [huggingface spaces](https://huggingface.co/spaces/GN1999/Financial-Advisor). 
2. Upload your monthly or yearly bank statement as a **CSV**.
3. Ask questions like "Summarize my spending" or "Which of these are deductible?"
4. The agent will run Python
